🧊 3D Model Guide

🖱 Mouse Controls

  • Left drag: Orbit camera
  • Right drag: Pan camera
  • Scroll: Zoom in/out
  • Click object: Select (shows axes + box)
  • Drag selected: Move object
  • Shift+drag selected: Stretch along axis
  • Scroll on selected: Resize
  • Ctrl+C/X/V: Copy / Cut / Paste
  • Delete: Remove selected
  • Esc: Deselect
  • Ctrl+S: Save / Ctrl+Shift+S: Save As
  • Ctrl+N: New project
  • Ctrl+Z: Undo

➕ Add Shape

  • Click ＋ Add → pick shape
  • Hover to position, scroll to resize
  • Click to place, Esc to cancel
  • Q/E R/F arrows to rotate preview

Select → move, resize, copy, paste. Merge all before STL export!

✏️ Sketch Guide

  • Draw with mouse or touch
  • Select tool texture from toolbar
  • Export → SVG for laser cutting

✋ Freeform Guide

🤲 Hand Gestures

  • 👌 Pinch: Select shape
  • ✊ Hold fist + move: Move selected shape
  • 👌 Push / pull: Scale shape
  • Release pinch: Drop in place
  • 🖐 Open palm: Rotate shape
  • 🖐🖐 Two palms: Zoom camera
  • 👌👌 Two pinch drag together: Move selected shape
  • 👌👌 Two pinch spread: Scale shape
  • ✊✊ Two fists apart: Stretch / grow
  • ✊✊ Two fists together: Squish / shrink
  • 🖐🖐 3 fingers + push: Sculpt surface in
  • 🖐🖐 3 fingers + pull: Sculpt surface out
  • ☝️ Point: Highlight faces
  • ✊ Fist: Freeze
  • 👌 Pinch empty: Deselect
